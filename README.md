#  Pocket Cart

A e-commerce web application built with React, Redux Toolkit, and Tailwind CSS. Browse products, manage your cart and wishlist, search in real time, and filter by category powered by the Fake Store API.

---

## Features

- **Product Browsing** — View all products or filter by category with active highlight
- **Real-Time Search** — Expandable search bar in the header with live filtering
- **Product Detail Page** — Full product info with quantity controls
- **Shopping Cart** — Add, remove, and update item quantities
- **Wishlist** — Save products for later
- **User Login** — Form validation with alphanumeric + special character password rules
- **Persistent State** — Cart, wishlist, and auth state saved in localStorage
- **Category Filter** — Fetch products by category with active button highlight
- **FAQ Accordion** — Animated accordion, one section open at a time
- **Animated Product Cards** — Image zoom, overlay, and quick action buttons on hover

---

##  Technologies used

| Technology | Purpose |
|---|---|
| React (Vite) | UI framework |
| Redux Toolkit | State management |
| react-redux | Connect Redux to React |
| React Router DOM | Client-side routing |
| Tailwind CSS | Utility-first styling |
| Axios | HTTP requests |
| Fake Store API | Product and category data |
